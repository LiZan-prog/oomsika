# Oomsika: Property Marketplace   **SwiftRent** is a nextjs app for lodging. Built for speed and a "native-app" feel.  ###  Features * **Dynamic Listings:** Filter by property type. * **Owner Portal:** Register and manage listings. * **Modern Stack:** Next.js 14, Tailwind CSS, Lucide Icons. * **Roadmap:** Co-renting matching.
